About Our Latest App

Rep’s Mate is deployed online and is accessible over the Internet (Cloud-based). Rep’s Mate Online Client can be accessible from full-fledged browsers, on laptops, desktops, and tablets. Rep’s Mate is also available as a mobile application under Android and iOS platforms.

Admin

The Administrator Client will be used by the application administrator to setup and update data at the organizational level

Client

Medical representatives will use the Online Client or the Mobile Application to report their Calls, Off-Fields, and Activities.

Report

Medical representatives, supervisors, managers, and the application administrator will use the Reporting System to generate reports at any time

Reports

Below is an overview of the available reports. The data being used is for demo purposes. For each of the below reports, the manager can select the medical representative to generate data for, as well as other parameters such as reporting period.

1. Effective Sales Line Target per Doctor
The report shows the number of planned visits per doctor versus the actual visits performed by the medical representative.
2. Samples Details per Doctor Specialty Report
The report shows the samples and the quantities distributed to each doctor. They are grouped per doctor specialty.
3. Samples Allocated to Medical Representative Report
The report shows the number of samples given by the company to the medical representative on specific dates.
4. Coverage Report
The report shows the number of calls conducted by medical representatives per category/profile of doctors. Categories in the below report are P1, P2, P3. They can as well be A1, A2, B1, B2, C1, and C2.
5. Customer Reach Report
The report shows the data in the customer coverage report but in percentages. (Data are for demo purposes are not real).
6. Effective Calls on Selected Brand Report
The report shows the number of calls which detail a certain brand. In the report below, there are 202 calls where the medical representative discussed the product “Aspirin”.
7. Samples Distributed Per Sales Representative Report
The report shows the number of samples distributed by medical representative in a given duration of time.
8. Giveaway Distributed Per Sales Representative Report
The report shows the number of giveaways/promotional items distributed by medical representative in a given duration of time.
9. Activities per MR Report
A sample report can be provided upon request.
10. Work Summary
The report summarizes the work done by each medical representative within a given period of time. It shows the number of calls, average of calls per day, number of activities, number of accompanies calls, etc..
11. Location of the Calls
The report shows the location of each call created by the medical representative as long as other data such as the Date of the Call, the Doctor visited. Time information can be added as well.
